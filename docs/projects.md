# Мої проєкти

У цьому розділі представлені ключові навчальні проєкти, над якими я працював під час навчання. Кожен проєкт відображає мій досвід у розробці алгоритмів та побудові користувацьких інтерфейсів.

---

## 1. Консольний менеджер завдань (Task Manager)
**Опис:** Програма для керування списком справ через консольний інтерфейс. Дозволяє додавати, видаляти та відмічати завдання як виконані.

* **Технології:** Мова C, робота з динамічною пам'яттю, файловий ввід/вивід.
* **Результат:** Надійна програма, що зберігає дані у текстовому файлі між сеансами роботи.

## 2. Персональний веб-сайт (Landing Page)
**Опис:** Розробка адаптивного односторінкового сайту-візитки з використанням сучасних методів верстки.

* **Технології:** HTML5, CSS3 (Flexbox/Grid), адаптивний дизайн (Media Queries).
* **Результат:** Повністю функціональна сторінка, яка коректно відображається на смартфонах, планшетах та десктопах.

---

## 3. Аналітична модель в MS Excel
**Опис:** Створення автоматизованої таблиці для розрахунку бюджету проєкту з візуалізацією витрат.

* **Технології:** Формули Excel, зведені таблиці (Pivot Tables), діаграми.
* **Результат:** Інструмент, що дозволяє швидко прогнозувати витрати на основі вхідних параметрів.
